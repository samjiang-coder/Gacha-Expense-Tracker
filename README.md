# Gacha Expense Tracker 🎮💰

一個結合「記帳」與「抽卡」趣味的網頁應用，讓你透過理財行為獲得抽卡代幣，增加存錢動力！

## 技術架構

- **前端**：HTML5 + [TailwindCSS](https://tailwindcss.com/)
- **後端**：Python [FastAPI](https://fastapi.tiangolo.com/) 框架
- **資料庫**：SQLite3（輕量級關聯式資料庫，內建於 Python）

## 檔案結構

- `main.py`：後端 API 伺服器（負責資料處理與抽卡邏輯）
- `index.html`：前端使用者介面（負責互動與呈現）
- `gacha.db`：SQLite 資料庫檔案（儲存記帳紀錄與代幣）
- `README.md`：專案說明文件（你現在正在看的地方）

## 如何啟動專案

### 前置準備

確保你的電腦已安裝 Python (建議 3.8+) 和 pip。

1.  **安裝依賴套件**：
    打開終端機或命令提示字元，切換到本專案資料夾，執行以下指令安裝 FastAPI 和 uvicorn：
    ```bash
    pip install fastapi uvicorn
    ```

### 啟動後端伺服器

在終端機中執行以下指令，啟動 FastAPI 伺服器：

```bash
uvicorn main:app --reload
```

- `--reload`：讓伺服器支援熱重載，修改程式碼後會自動重新啟動。
- 執行成功後，終端機會顯示類似 `Uvicorn running on http://127.0.0.1:8000 (Press CTRL+C to quit)` 的訊息。

### 開啟前端網頁

1.  用瀏覽器開啟 `index.html` 檔案。
2.  **首次使用時**：請先點擊「重新整理」按鈕，確保與後端伺服器連線正常。

## 功能說明

### 記帳功能

- 在左側區塊輸入「品項」和「金額」。
- 點擊「送出記帳」按鈕。
- **邏輯**：系統會將金額換算成代幣（每 10 元 = 1 代幣），並加到你的代幣餘額中。
- 歷史帳目會即時顯示在下方。

### 抽卡功能

- 在右側區塊點擊「抽取盲盒！」按鈕。
- **條件**：每次抽取需要 20 代幣。
- **獎勵**：
  - **SSR** (5% 機率)：頂級省錢大師卡 💎
  - **SR** (25% 機率)：節約小能手卡 ✨
  - **R** (70% 機率)：平凡打工人卡 👤
- 抽中的卡片與結果會顯示在畫面上，並同步更新代幣餘額。

## 安全與功能亮點

本專案已針對基礎資安進行防護：

1.  **輸入驗證**：
    - 後端嚴格檢查金額是否大於 0，防止負數攻擊。
2.  **SQL 注入防護**：
    - 採用參數化查詢 (`?`)，杜絕惡意 SQL 語法注入。
3.  **後端強制運算**：
    - 抽卡機率與代幣扣除都在後端執行，防止前端篡改數值作弊。
4.  **跨域存取 (CORS)**：
    - 允許前端與後端在不同來源正常通訊。
